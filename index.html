What is Google Drive Index ?

An Index is nothing but a list of Files or Documents stored on a Server , in the same way Google Drive Index is a list of Files and Documents stored in your Google Drive.

Google Drive Index uses the Google Drive API to access the files on user's Google Drive and lists them statically on a custom Domain.

Features

  • Security

    Google Drive Index is encrypted using 256-Bit SSL, the same security protocol used on Gmail and other Google services.

  • Privacy

    None of the Files are actually stored on our Servers. All the Files are stored in your own Google Drive.

  • Link Sharing

    Sharing is easier than ever , the Links are Static so just copy and share Links to your contacts.

  • Unlimited Transfers

    Since we are using the Google Drive API there are no restriction on number of times a file is downloaded.

  • No Bandwith Limits

    There are no Bandwidth restrictions on downloads.

  • 99.9 % Uptime

    We use Premium VPS's located in multiple locations , so there are no server downtimes.
